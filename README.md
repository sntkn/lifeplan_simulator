# loan_simulator

## 概要

年間の収入、支出を算出し、将来いくら手元に残るか計算するアプリです。
インフレ率、投資リターンも考慮します。

## 入力項目

### 基本項目

- 年齢
- インフレ率: 例（1.02
- 投資名目リターン: 例（1.07
- 株を取り崩す際の税率: 例（1.1
- 仮想通貨を取り崩す際の税率: 例（1.3

### 資産初期値

- 株保有額: 例（125,000,000
- 仮想通貨保有額: 例（50,000,000
- 現金: 例（50,000,000

### 年間支出項目

- 生活費: 例（3,000,000
- 娯楽費: 例（1,000,000
- 住宅維持費: 例（600,000
- 医療・介護費: 例（750,000
- 住宅ローン: 例（210,000

### 年間収入項目

- 給与所得: 例（4,500,000
- 不動産所得: 例（1,000,000

## 運用ルール

- 収入の合計から支出の合計を引き、現金に反映する
  - 収支がプラスの場合
    - 現金が規定の金額（例 20,000,000）を上回ったら、株に上乗せする
  - 収支がマイナスの場合
    - 現金残高が下限 例（15,000,000）を下回った場合、仮想通貨から取り崩す
      - 取り崩す際は規定の税率（例 30%）を上乗せする
    - 仮想通貨残高の下限 例（10,000,000）を下回った場合、株から取り崩す
      - 取り崩す際は規定の税率（例 30%）を上乗せする
- 75歳からは娯楽費を毎年10%ずつ削減する
- 75歳からは医療費・介護費が支出に加わる。それまでは0とする
- インフレは、仮想通貨、生活費、娯楽費、住宅維持費、医療・介護費、不動産所得に毎年かかる
- 給与所得は増えずに55歳まで毎年もらうものとする
- 住宅ローンは一定で30年間とする

## 表示項目

年を行、以下項目を列として100歳まで表示する

- 給与所得
- 不動産所得
- 収入合計
- 生活費
- 娯楽費
- 住宅維持費
- 医療・介護費
- 住宅ローン
- 支出合計
- 収入-支出
