# Functional Specification

## 入力項目

### シミュレーション設定

- シミュレーション方法：モンテカルロ / ヒストリカル
- シミュレーション回数（モンテカルロのみ）
- 株式リターンの地域：S&P500、日経平均、世界株式（ヒストリカルのみ）
- インフレ率の地域：日本、米国、世界（ヒストリカルのみ）

### 基本項目

- 初期年齢 / 終了年齢（最大 120 歳）
- モンテカルロ法のパラメータ：インフレ率、株式リターン・リスク、仮想通貨リターン・リスク
- 売却コスト：株式・仮想通貨の税/手数料（例: 0.1 = 10%）
- 現金・株式・仮想通貨の上下限
- 現金不足時の優先取り崩し資産（仮想通貨 / 株式 / ランダム）
- リタイア年齢、ローン年数、医療/介護費発生年齢、娯楽費減少開始年齢と減少率

### 資産初期値（万円）

- 株式、仮想通貨、現金

### 年間支出（万円）

- 生活費 / 娯楽費 / 住宅維持費 / 医療・介護費 / 住宅ローン

### 年間収入（万円）

- 給与所得 / 不動産所得

## 運用・収支ロジック

1. 資産運用
   - 株式・仮想通貨は設定リターンとリスクで推移
   - 現金はリターンなし
2. 収支計算
   - 収入−支出を現金に反映
   - 収支プラスで現金が上限超過 → 優先設定に従い自動投資
   - 収支マイナスで現金が下限割れ → 優先設定に従い資産取り崩し
3. 取り崩しルール
   - 優先資産（仮想通貨 / 株式 / ランダム）順で売却
   - 税率・手数料を控除し、下限割れ資産は他資産で補填
   - 両資産が下限を割った場合は現金をマイナス許容で補填

## 年齢による変化

- リタイア年齢までは給与所得を計上し、翌年以降は 0
- 設定年齢から医療・介護費を支出に追加
- 設定年齢から娯楽費を減少率に沿って毎年削減

## 表示仕様

- 結果テーブル：年齢と各パーセンタイル（10/25/50/75/90）、資産内訳（株式/仮想通貨/現金中央値）
- 資産推移グラフ：資産額と構成比を描画し、パーセンタイル線を重ねる

## シミュレーション期間

- モンテカルロ法：最大 100 年間
- ヒストリカル法：最大 100 年間（30 年以内は実データ、30 年超は 30 年分を循環使用）
