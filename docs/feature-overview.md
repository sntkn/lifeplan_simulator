# Feature Overview

## 概要

ライフプランシミュレーターは年間の収入・支出・資産配分を入力し、将来の手元資金を推定する React + TypeScript 製アプリです。インフレ率や投資リターンを考慮し、複数手法で資産推移を比較できます。

## シミュレーション手法

### モンテカルロ法

- 確率的シミュレーションでランダム要素を取り入れる
- 期待リターンとリスク（標準偏差）に基づき理論値を多数回サンプリング
- 未来の不確実性を統計的な分布として可視化

### ヒストリカル法

- 1996-2025 の過去 30 年データを複数の開始年で走査
- 実際の市場変動を再現することでシナリオ別の推移を確認
- 30 年超の長期は 30 年分のデータを循環利用し、100 年を上限
- 2014 年以前の仮想通貨リターンは株式リターンを代用

## 入力・出力のハイライト

- シミュレーション設定（手法、回数、地域別リターン/インフレ、リタイア年齢など）
- 資産初期値（株・仮想通貨・現金）と年間収入/支出の細目
- 結果テーブル：年齢ごとの資産中央値、各パーセンタイル、資産内訳
- グラフ表示：年齢を横軸、資産額と構成比を縦軸にプロット

## 特徴的な挙動

- インフレ適用：生活費や不動産所得、資産価値を設定インフレ率で調整
- 現金の上下限と優先取り崩し資産のロジックを実装し、余剰時は自動投資、不足時は税/手数料込みで取り崩し
- 娯楽費・医療費など年齢で変動する支出ルールをサポート
- 設定の保存/読み込み/削除に対応し、ユーザーごとのプリセット管理が可能
