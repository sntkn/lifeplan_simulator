
# 更新履歴

## v1.2.0 (2025/11/8)

### BREAKING CHANGES

#### パラメータの仕様（追記: 税率の取り扱い）

- stockTaxRate / cryptoTaxRate:
  - 税率は fraction（小数）で指定します。例: 0.10 = 10%、0.13 = 13%。
  - 値は 0.0 以上 1.0 未満を想定します（1.0 = 100% は不可）。
  - 古い設定で 1.1 や 1.13 のような「multiplier（1 + 税率）」形式を使っている場合は、fraction に変換してください（例: 1.10 -> 0.10、1.13 -> 0.13）。
  - コードは fraction を前提に動作します。README と既存設定を合わせてください。

## v1.1.0 (2025/8/2)

- **現金上限超過時の処理改善**: 現金が上限を超えた場合、「現金不足時の優先取り崩し資産」設定に基づいて投資先を決定するように変更
- **ランダム選択オプション追加**: 「現金不足時の優先取り崩し資産」にランダムオプションを追加
- **ランダム選択の条件**: 両方の資産が下限以上の場合のみランダムで決定、そうでなければ適切な資産を選択
- **コード改善**: if-else文をswitch文に変更してより読みやすく保守しやすいコードに改善
